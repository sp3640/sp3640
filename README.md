```
                    ┌─────────────────────────────────────────┐
                    │            DevOps Lifecycle              │
                    └─────────────────────────────────────────┘

  📋 Plan  ──►  🖊️ Code  ──►  🔨 Build  ──►  ✅ Test
                                                    │
  📡 Monitor  ◄──  🚀 Deploy  ◄──  📦 Release  ◄──┘
        │
        └──►  🔁 Feedback  ──►  📋 Plan  (repeat)

  Tools I use across the lifecycle:
  ─────────────────────────────────────────────────
  Plan    │ Jira, GitHub Issues
  Code    │ Git, GitHub, GitLab
  Build   │ Jenkins, GitHub Actions, GitLab CI
  Release │ Terraform, Ansible
  Deploy  │ AWS / Azure / GCP
  Monitor │ Prometheus, Grafana
  ─────────────────────────────────────────────────
```
